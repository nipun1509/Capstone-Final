# 🚀 EmoLearn
AI-Powered Emotion-Aware Learning Platform

---

## 📌 Overview

EmoLearn is an intelligent learning platform that enhances digital education by detecting student emotions and adapting teaching methods accordingly.

It uses AI and NLP to understand how a student feels (confused, frustrated, motivated) and adjusts explanations in real time through an AI tutor, creating a more personalized and effective learning experience.

---

## ✨ Features

### 🤖 AI Tutor Chatbot
- Interactive chatbot for real-time learning  
- Explains concepts step-by-step  
- Acts like a personal tutor  

### 🧠 Emotion Detection
- Detects emotions from student input  
- Supports: Confused, Frustrated, Motivated, Neutral  

### 🎯 Adaptive Learning
- Dynamically adjusts explanation style  
- Simplifies or deepens content based on emotion  

### 📚 Courses Module
- Browse and select subjects  
- Start learning instantly  

### 📊 Progress Dashboard
- Track learning progress  
- View performance insights  

### 🏆 Achievements System
- Unlock badges and milestones  
- Gamified learning experience  

### ⚙️ Settings Page
- Customize learning preferences  
- Control AI tutor behavior  

---

## 🧠 How It Works

User Input → Emotion Detection → Prompt Builder → AI Model (OpenRouter) → Adaptive Response → Chat Interface

---

## 🎯 Emotion-Based Adaptation

| Emotion     | System Behavior                          |
|------------|------------------------------------------|
| Frustrated | Simplifies explanation and encourages    |
| Confused   | Provides step-by-step clarification      |
| Motivated  | Gives slightly advanced explanation      |
| Neutral    | Standard explanation                     |

---

## 🏗️ Tech Stack

### Frontend
- React.js  
- Tailwind CSS  
- Framer Motion  

### Backend
- Node.js  
- Express.js  

### AI / NLP
- OpenRouter API  
- LLM Models (Qwen / others)  
- Rule-based Emotion Detection

## ⚙️ Setup & Run

### 1. Clone Repository
git clone https://github.com/your-username/emolearn.git

### 2. Navigate to Project
cd emolearn

### 3. Install Dependencies
npm install

### 4. Run Backend
node server.js

### 5. Run Frontend
npm run dev
